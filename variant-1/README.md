# WordPress + MySQL + Redis (Docker Compose)

## Описание стека
Этот проект разворачивает CMS WordPress с базой данных MySQL и кэшированием Redis в Docker-контейнерах.

**Сервисы:**
- **WordPress 6.5** — CMS на порту 8080
- **MySQL 8.0** — база данных
- **Redis 7-alpine** — кэш объектов
